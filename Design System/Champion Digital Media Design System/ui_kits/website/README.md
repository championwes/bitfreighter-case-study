# Champion Digital Media — Website UI Kit

A high-fidelity recreation of championdigitalmedia.com, broken into reusable React components. Open `index.html` to see the click-thru.

## Files

- `index.html` — full marketing page with nav, hero, manifesto, work grid, dark CTA, contact modal, footer.
- `components.jsx` — all components, exported to `window` so the page can mount them.

## Notes

- Designed at 1440px (the brand's design canvas). The page scales down responsively for the preview.
- PODIUM Sharp is substituted with Big Shoulders Display Italic — see root README.
- Photography is stand-in (warm-cool gradient blocks). Real images go into `/assets/photos/` and replace the gradients.
- The contact button uses the signature circled-word gesture and is wired to open a modal.
